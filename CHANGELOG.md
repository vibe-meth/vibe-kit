## [0.1.0] - 2024-01-02

### Added
- Initial release of VibeCoder
- Core CLI: `vibe init`, `vibe check`, `vibe version`
- Six command templates: plan, tasks, execute, clarify, insight, review
- Bash and PowerShell script support
- Dev container with AI agent pre-installation
- Support for 19+ AI agents (Claude, Amp, Cursor, Copilot, etc.)
- Comprehensive documentation and quick start guide
- Phase-based planning with confidence scoring
- Real-time task execution tracking
- Blocker detection and management
